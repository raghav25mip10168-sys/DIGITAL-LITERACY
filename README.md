# DIGITAL-LITERACY
This project focuses on developing essential digital skills needed in today’s world. It covers topics like understanding digital tools, safe internet practices, email etiquette, and building a professional online presence. The aim is to help students use technology responsibly, effectively, and securely for academic and professional growth.

Project Overview

This repository contains my Digital Literacy Portfolio created as part of the CSE0001 course at VIT Bhopal University.

As a Student Digital Ambassador, the objective of this project was to explore digital tools, build a professional online presence, practice coding and collaboration platforms, understand professional communication, and create awareness about cybercrime prevention.

This repository includes five tasks covering all modules of the course along with a detailed project report.
Module-wise Summary
Task 1 – Digital Literacy Infographic
